# react-redux-server

App example of create-react-app with react-redux, working on express.js server

## Installation
```
$ npm install
$ cd app 
$ npm install

or

$ make install 
```
## Running Dev Server
```
$ cd app
$ npm start

or

$ make dev 
```
## Running Production Server
```
$ cd app 
$ npm run build
$ cd .. npm start

or

$ make production
```
## Build
```
$ cd app 
$ npm run build

or

$ make build
```
